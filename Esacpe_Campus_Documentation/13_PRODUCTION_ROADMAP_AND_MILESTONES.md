# PROJECT GOAL

Deliver a complete, polished, playable horror game.

Target Platform:

Windows PC

Distribution:

itch.io

Target Playtime:

60–90 Minutes

Target Genre:

Psychological Horror
Investigation Horror
Narrative Horror

Target Release:

Within 12 Weeks

# SUCCESS CRITERIA

Project considered successful if:

✓ Game can be completed

✓ No critical bugs

✓ Stable 60 FPS

✓ All endings functional

✓ All puzzles solvable

✓ Horror pacing functional

✓ itch.io build uploaded

✓ Documentation complete

# BUG PRIORITY SYSTEM
########################################################

P0

Critical

Examples:

Cannot finish game

Save corruption

Crash

---

Fix Immediately

---

P1

High

Examples:

Puzzle break

Major event failure

---

Fix Before Next Milestone

---

P2

Medium

Examples:

Visual bugs

Audio bugs

---

Fix Before Beta

---

P3

Low

Examples:

Typos

Minor clipping

---

Fix Before Launch
