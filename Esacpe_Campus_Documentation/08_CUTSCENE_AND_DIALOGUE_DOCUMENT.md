Environmental Storytelling First

Dialogue Second

- Dokumen

- Echo Victims

# DIALOGUE RULES

- Tidak berteriak terus-menerus
- Tidak terdengar seperti villain

########################################################
# AI AGENT RESTRICTIONS
########################################################

AI Agent must never:

- Rewrite Raka's motivation
- Turn Raka into pure evil
- Add comedy dialogue
- Add combat dialogue
- Add action hero dialogue

---

AI Agent must preserve:

- Tragic horror
- Psychological horror
- Academic horror
